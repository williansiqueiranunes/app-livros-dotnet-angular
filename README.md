# Tecnologias usadas
### back-end
- .NET 8
- Entity Framework
### Front-end
- Antular 19
- Angular Material
---

# Como rodar o projeto

Clone o projeto do GitHub
```sh
git clone https://github.com/williansiqueiranunes/app-livros-dotnet-angular.git
```


## Rodando Back-end

No terminal, entre na pasta `Livraria.Api` e rode os comandos:
```sh
dotnet restore

dotnet build

dotnet run
```
O swegger está rodando em:
```sh
localhost:5135/swagge
```

## Rodando Front-end

No terminal, entre na pasta `Livraria.Web` e rode os comandos:
```sh
npm i

npm start
```
O Angular está rodando em:
```sh
127.0.0.1:62642
```
